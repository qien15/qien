Introduction of Paragraph Tag

In HTML, paragraphs are defined with the p tag.
A paragraph element in an Html document is made up of content
such as text, images, or other content
which appease between an opening p tag and closing p tag.

p tag is the HTML tag for a paragraph. Adding the p tag to an HTML doc, followed by text, will create a paragraph on your page.
Adding the closing p tag will end the paragraph.
